Ready-to-import Android Studio project scaffold that mirrors the MADD UI report and assignment requirements.

- Room entities/DAO/DB, asset seeding stubs

- Seed JSONs: attractions, events, posts

## Import
1. Open Android Studio → **Open** → select the `TripMate` folder.
2. Let Gradle sync. If prompted, install KSP and Kotlin plugins.
3. Run the **TripMateActivity**.
